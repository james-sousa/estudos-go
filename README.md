# estudos-go
Go Studies Repository

Este repositório tem como objetivo documentar minha jornada de aprendizado na linguagem Go (Golang), indo do nível básico ao avançado, com exemplos práticos, projetos e anotações.

 Objetivos
Aprender os fundamentos da linguagem Go
Entender conceitos de concorrência e paralelismo
Desenvolver aplicações reais
Explorar boas práticas e padrões de projeto
Preparação para uso profissional e projetos escaláveis

 Conteúdo
 Básico
Sintaxe da linguagem
Tipos de dados
Estruturas de controle (if, for, switch)
Funções
Arrays, slices e maps
 Intermediário
Structs e métodos
Interfaces
Tratamento de erros
Organização de código com pacotes
Leitura e escrita de arquivos
 Avançado
Goroutines
Channels
Padrões de concorrência
Context API
Testes e benchmarks
